# Customer Churn Prediction
Customer Churn Prediction using Machine Learning - Internship Task 2

## Project Overview
This project predicts whether a telecom customer will churn using Machine Learning.

## Dataset
- Name: Telco Customer Churn
- Records: 7,043 customers | 21 features

## Models Used
- Decision Tree
- Random Forest

## Files
- churn_prediction.py — Python source code
- Telco-Customer-Churn_1.csv — Original dataset
- churn_cleaned.csv — Cleaned dataset

## How to Run
1. Install libraries: pip install pandas matplotlib seaborn scikit-learn
2. Place dataset in same folder as the code
3. Run: python churn_prediction.py
